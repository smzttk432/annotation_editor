汎用注釈ドキュメントメーカー

画像に番号と説明を付けて、単一HTMLで配布・再編集。
ブラウザだけで動作／オフライン可／MIT

エディタを開く GitHubで見る

情報は外部へ送信しません。作成したドキュメントは1つのHTMLに保存され、後から再編集できます。

主な特徴

  • ブラウザだけで動作、インストール不要
  • 番号クリックで説明・表をハイライト
  • HTMLひとつで配布・閲覧・再編集が可能
  • 自由記述、テンプレート機能付き
  • ローカル完結、オフライン利用可

使い方

  1. 「エディタを開く」から編集画面へ
  2. 画像を読み込み、クリックで番号を追加
  3. 右欄や表に説明を入力
  4. 保存してHTMLファイルとして配布（後から再編集できます）

動作環境：Chrome / Edge / Firefox 最新版（PC推奨）
